#glitchbone-starter

Glitchbone Frontend Starter Kit

##Installation

```sh
git clone --depth=1 https://github.com/Glitchbone/starter.git my_project
cd my_project
rm -rf .git
npm install
bower install
```

##Usage

Launch development server:
```sh
grunt serve
```

Build project in dist folder:
```sh
grunt build
```
